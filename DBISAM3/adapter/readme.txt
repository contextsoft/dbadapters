INSTALLATION INSTRUCTIONS
=========================

Copy this dll to the 'adapters' subfolder of Context Database Designer 
(default: C:\Program Files\Context Software\DBDesigner\adapters\).
Run Database Designer and add the adapter using Tools | Options dialog
(see Adapters page).
